# OPC C# Client

This repository contains:
* The `PositionClient.cs` C# OPC UA client is designed to allow constant positional updates with an OPC server.
* The `UIClient.cs` C# OPC UA client is designed to allow interact with a front-end to switch between robot targets (i.e. Printer, Arm, Dog) and then communicate with an OPC server.
* The `server.py` OPC UA server is used for testing purposes.
